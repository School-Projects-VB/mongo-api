# ⚙️ MongoDB API
Mongo API for lessons

### Dependencies
- [npm](https://www.npmjs.com/)

### Links
- [API](http://localhost:3000)
- [Documentation](http://localhost:3000/api-docs)

## Linux & MacOS

### Installation
1. Configure your MongoDB server
   > You can also configure a docker

2. Create a `.env` file at the root of the project
    ```dotenv
    PORT=your_port
    HOST=your_host
    DATABASE=your_database
    SECRET=your_secret
    ```

### Getting started

- Start server
    ```bash
    make run
    ```

## Windows

### Installation
1. Configure your MongoDB server 
    > You can also configure a docker

2. Create a `.env` file at the root of the project
    ```dotenv
    PORT=your_port
    HOST=your_host
    DATABASE=your_database
    SECRET=your_secret
    ```

3. Install npm dependencies
    ```bash
    npm install
    ```

### Getting started

- Start server
    ```bash
    npm start
    ```
